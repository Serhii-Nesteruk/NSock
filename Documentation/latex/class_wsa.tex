\doxysubsection{Detailed Description}
Handles the initialization and cleanup of Win\+Sock using the Singleton pattern. 

The \doxylink{class_wsa}{Wsa} class ensures that Win\+Sock is properly initialized and cleaned up using the Singleton pattern. It provides a single instance that can be accessed globally and guarantees that Win\+Sock is initialized only once during the application\textquotesingle{}s lifetime.
