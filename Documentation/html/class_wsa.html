Detailed Description

Handles the initialization and cleanup of WinSock using the Singleton pattern.

The Wsa class ensures that WinSock is properly initialized and cleaned up using the Singleton pattern. It provides a single instance that can be accessed globally and guarantees that WinSock is initialized only once during the application's lifetime.
